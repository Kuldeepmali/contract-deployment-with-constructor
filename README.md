# Contract Deployment With Constructor

This package helps in understanding the steps to deploy a contract with constructor parameters using ethereum-js, solc and web3 packages by encoding the bytecode

## Dependencies

| Dependency | Version |
| ---------- | ------- |
| Node       | >11.0.0 |
| NPM        | >6.13.4 |
| solc       | 0.6.12  |
| web3       | 1.2.11  |
